# Telegram-Nft-Fishing-Full-code
Телеграм бот который позволяет создавать сделки и оставлять других пользователей без nft подарков.

Бот работает очень просто. Пользователь создает сделку -> делится ей с вами -> вы вступаете -> пользователь заполняет данные о своем NFT -> вам приходить сообщение с фейк оплатой -> после того как вы произвели фейк оплату, пользователю отправляется сообщение о том, что пора скидывать NFT

Полный и бесплатный доступ к коду 

Замените ADMIN_ID и TOKEN на ваши и все, бот готов к запуску!
